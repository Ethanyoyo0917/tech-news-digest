## 🤖 AI Agent

- **Meta Manus 桌面端正式发布，AI Agent 进入本地化时代** — Meta 宣布 Manus 桌面应用登陆 Windows PC 与 Mac，旗下"My Computer Agent"可直接管理本地文件、发送邮件、整理文件夹乃至构建应用程序，标志着 AI Agent 从云端向本地设备的大规模迁移。[PCMag](https://www.pcmag.com/news/metas-manus-launches-desktop-app-with-ai-agent-for-tasks)

- **西门子 Fuse EDA AI Agent：半导体设计进入全流程自主化** — 西门子数字发布 Fuse EDA AI Agent，基于 RAG 管道与多模态 EDA 数据湖，支持从架构探索到 RTL 编码、布局布线的全生命周期自动化，整合 Questa、Aprisa、Calibre 等主流 EDA 工具，并支持 NVIDIA Agent Toolkit。[Electronics Weekly](https://www.electronicsweekly.com/news/design/eda-and-ip/siemens-digital-launches-autonomous-ai-agent-2026-03/)

- **Galileo Agent Control：开源 AI Agent 治理层正式发布** — 3 月 13 日，Galileo 推出 Agent Control，为 AI Agent 建立通用行为标准与安全护栏，配备 Runtime Mitigation 功能，支持在不停机情况下动态调整 Agent 安全策略。[Boston Institute of Analytics](https://bostoninstituteofanalytics.org/blog/agentic-ai-news-roundup-7-13-march-2026-market-growth-enterprise-adoption-new-ai-agents/)

- **Agentic AI 市场预计 2034 年达 1390 亿美元，72% 全球 2000 强已部署** — 据最新报告，Agentic AI 市场将从 2026 年初的 91.4 亿美元增长至 2034 年的 1390 亿美元（CAGR 40.5%），"AgentOps"基础设施成为 VC 新风口，但数据准备度不足仍是企业规模化落地的最大瓶颈。[Boston Institute of Analytics](https://bostoninstituteofanalytics.org/blog/agentic-ai-news-roundup-7-13-march-2026-market-growth-enterprise-adoption-new-ai-agents/)

- **Salesforce 推出六款医疗健康 AI Agent** — Salesforce 在 Agentforce Health 平台上新增六款自主 Agent，包括传染病实时追踪 Agent、保险核验 Agent 与临床文档 Agent，标志着从通用聊天机器人向领域深度专业 Agent 的转型趋势。[Boston Institute of Analytics](https://bostoninstituteofanalytics.org/blog/agentic-ai-news-roundup-7-13-march-2026-market-growth-enterprise-adoption-new-ai-agents/)
